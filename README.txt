Welcome.


This script was designed for downloading every single course class from www.codewithmosh.com (4000 video files, 350 GB). 
All video files are in different web pages.

Right below is a step-by-step guide on how to put it to work.

1) Get your cookies and headers from the browser and paste in the headers.py file.

2) Export environmental variables USER and PASSWORD.

set (windows) or export(linux/GNU) USER=email
set (windows) or export(linux/GNU) PASSWORD=password

3) Run the script

4) The files will be downloaded to your Downloads folder, separated by course and chapter.
